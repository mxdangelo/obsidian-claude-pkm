# Useful Links

Practical bookmarks to find again when needed — not sources to distill, just links you might
want to reopen. `/capture` proposes a line here when a URL is a reference page (a tool's
homepage, a dashboard, docs) rather than a source with a thesis.

- [Manage vaults in Obsidian](https://obsidian.md/help/manage-vaults) — official help: create, rename, and move vaults; the config lives in `.obsidian`, copyable between vaults
